# TODOAPP

* Ruby on Railsでタスク管理アプリを作成しました。

## 使い方

* 「CREATE NEW TASK」ボタンでタスクの生成
* タスク右側の削除ボタンでタスクの削除

## 使用技術

* Ruby
* HTML
* CSS
* JavaScript
* Bootstrap
* Ruby on Rails
* SQLite
* Heroku
